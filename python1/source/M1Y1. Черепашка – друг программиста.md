# Переменная

- **Переменная** – это **именованная ячейка памяти**, в которой хранятся какие-то **данные**
  <img src="https://cdn-user84060.skyeng.ru/uploads/6281ebe31e22c597983240.png" width="25%" align="right">
```python
var = 5 # где var – имя переменной, = – оператор присваивания, 5 – значение переменной
```
## Тип данных
- **Тип данных** – это *характеристика* переменной. Каждый тип данных обладает:
  1. Уникальным **диапазоном значений**
  2. **Операциями**, которые можно производить между объектами этого и других типов данных

| Тип данных | Значения         | Пример         |
| ---------- | ---------------- | -------------- |
| `int`      | Целые числа      | `var = 5`      |
| `str`      | Набор символов   | `s = "строка"` |
| `float`    | Нецелые числа    | `var = 5.5`    |
| `bool`     | `True` и `False` | `b = True`     |
| `NoneType` | `None`           | `n = None`     |
# Интерпретатор
**Транслятор** – это *переводчик* с человеческого языка (языка программирования) на машинный. Есть два вида транслятора:
- **Интерпретатор** – переводит в режиме реального времени
- **Компилятор** – переводит всё и сразу
# Модуль Turtle
- **Пиксель** – это **частичка** экрана
- **Растр** – это **набор пикселей**
- **Модуль** – это **файл** с расширением `.py`
- **Turtle** – это *модуль* стандартной библиотеки Python
```python
from turtle import * # импортируем все команды из модуля

# 
forward(50) # идем вперед на 50 пикселей
back(50)    # идем назад на 50 пикселей
left(90)    # поворачиваемся влево на 90 градусов
right(90)   # поворачиваемся вправо на 90 градусов
reset()
```
